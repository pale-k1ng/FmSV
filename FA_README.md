## 🚀 روش استفاده

### مرحله ۱: اجرای برنامه
```bash
./FmSV_1.0.exe  # ویندوز
```

### مرحله ۲: تنظیمات سرور
از شما خواسته می‌شود تا وارد کنید:

| عنوان انگلیسی | توضیح فارسی |
|---------------|-------------|
| **Server Name** | نام نمایشی سرور شما |
| **Gamemode** | حالت بازی (`survival` یا `creative`، پیش‌فرض: survival) |
| **Max Players** | حداکثر تعداد بازیکنان (پیش‌فرض: ۲۰) |
| **Difficulty** | سطح سختی (`easy`، `normal` یا `hard`، پیش‌فرض: normal) |
| **Online Mode** | حالت آنلاین (`true` یا `false`، پیش‌فرض: true) |
| **View Distance** | فاصله دید بر حسب چانک (پیش‌فرض: ۱۰) |
| **Simulation Distance** | فاصله شبیه‌سازی بر حسب چانک (پیش‌فرض: ۱۰) |
| **Level Seed** | سید (دانه) دنیا (اختیاری) |
| **Hardcore** | حالت هاردکور (`true` یا `false`، پیش‌فرض: false) |

### مرحله ۳: انتخاب نوع سرور
از بین ۸ نوع سرور موجود، یکی را انتخاب کنید (۱-۸)

### مرحله ۴: انتخاب نسخه ماینکرفت
- تمام نسخه‌های موجود را مشاهده کنید
- شماره مربوط به نسخه موردنظر را تایپ کنید

### مرحله ۵: تخصیص رم
میزان رم را مشخص کنید (مثلاً `2G`، `4G`، `8G`)

### مرحله ۶: اجرا!
- انتخاب کنید که سرور را فوراً اجرا کنید یا بعداً
- اگر بعداً، فایل `start.bat` را اجرا کنید تا سرور شما شروع به کار کند

## ⚙️ تنظیمات

### فایل‌های تولیدشده
پس از راه‌اندازی، پوشه سرور شما شامل موارد زیر خواهد بود:

```
📁 نام_سرور_شما_نوع_نسخه/
├── server.jar              # فایل اجرایی سرور
├── start.bat               # اسکریپت اجرا (ویندوز)
├── server.properties       # تنظیمات سرور
├── eula.txt                # توافق‌نامه EULA (به‌طور خودکار تأیید شده)
├── fmsv.fmsv              # متادیتای سرور
```

### تنظیمات سرور
تنظیمات زیر به‌طور خودکار پیکربندی می‌شوند:
```properties
max-players=          # حداکثر تعداد بازیکنان
gamemode=             # حالت بازی (survival/creative)
difficulty=           # سطح سختی (easy/normal/hard)
hardcore=             # حالت هاردکور (true/false)
level-seed=           # سید دنیا
online-mode=          # حالت آنلاین (true/false)
view-distance=        # فاصله دید بر حسب چانک
simulation-distance=  # فاصله شبیه‌سازی بر حسب چانک
```

## 🐛 عیب‌یابی

### خطای "Java not found"
- برنامه به‌طور خودکار جاوا را نصب می‌کند
- در صورت مشکل، جاوا ۱۷ را به‌صورت دستی از [وب‌سایت رسمی](https://www.oracle.com/java/technologies/downloads/) نصب کنید

### خطای "Failed to download server"
- اتصال اینترنت خود را بررسی کنید
- ممکن است سرورهای موجنگ موقتاً در دسترس نباشند، چند دقیقه بعد دوباره امتحان کنید

### خطای "Port already in use"
- پورت ۲۵۵۶۵ ممکن است توسط برنامه دیگری استفاده شود
- در فایل `server.properties` پورت را تغییر دهید
